# WebOCR
Создана для запуска минимальной ии модели для распознования английского текста, работает в браузере через js и WebAsembly. скопилированы из Rust

# Использование
скачайте из www или релиза файлы WebOCR.js, WebOCR_bg.wasm
И саму модель model.onnx ‛rust/assets‛

# Предупреждение ⚠️
Сейчас модель может плохо работать, обучалась через github action 30000 файлов датасет и 30 эпох.
для создания нормальной модели можете поставить в action числа выше или переписать модель
